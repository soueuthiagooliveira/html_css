TAGS

O que são as tags?

Existem muitas tags em HTML, cada uma com sua função específica.
Algumas das tags mais comuns e amplamente utilizadas em HTML incluem:

html: Define o início e o fim de um documento HTML.
head: Contém informações sobre o documento, como título, links para estilos e scripts.
title: Define o título do documento exibido na barra de título do navegador.
body: Contém o conteúdo principal do documento, como texto, imagens, links e outros elementos.
h1, h2, h3, h4, h5, e h6: Definem cabeçalhos de diferentes níveis de importância.
p: Define um parágrafo de texto.
a: Define um link para outra página ou recurso.
img: Insere uma imagem na página.
ul: Define uma lista não ordenada, sua indicação são apenas por bolinhas
ol: Define uma lista ordenada, sua indicação é por sequêcia numérica.
li: Define um item de lista dentro de ul ou ol.
button: Inclui um botão na página.
table: Inclui uma tabela na página.

Essas são apenas algumas das tags mais comuns em HTML.
Existem muitas outras tags disponíveis para diferentes fins e necessidades de formatação de documentos HTML.
